## About This Certification

The Terraform Associate certification validates proficiency in infrastructure as code concepts and the ability to use HashiCorp Terraform to manage cloud infrastructure across multiple providers.

## Topics Covered

- Infrastructure as Code concepts and workflows
- Terraform CLI, configuration language (HCL), and state management
- Module composition and reusability patterns
- Provider configuration for multi-cloud deployments
- Terraform Cloud and Enterprise collaboration features
